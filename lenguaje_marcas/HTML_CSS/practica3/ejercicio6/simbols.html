Entitats HTML

Alguns caràcters estan resrvats en HTML.

Si utilitzeu < o> al vostre text, el navegador podria cobfondre'ls amb els de les etiquetes.

Les entitas de caràcters s'utlitzen per mostrar caràcters reservats en HTML.

Molts símbols matemàtics, tècnics i de moneda no estan presents en un teclat normal. Per afegir aquests símbols a una pàgina HTML, podeu utilitzar un nom d'entitat HTML.

Si no hi ha cap nom de l'entitat, podeu utilitzar un número d'entitat, una referència decimal o hexadecimal.
