FORMAT DE TEXT 1

Aquest text està dissenyat amb algunes de les propietats de format de text. L'encapçalament utilitza les propietats de text-align, text-transform, and color. El paràgraf està indentat, alineat i s'especifica l'espai entre caràcters. El subratllat s'elimina d'aquest enllaç "Canvia el format tu mateix".
